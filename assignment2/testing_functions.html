Part 4: Selecting Features and Training a Model¶

Now that I've created discretized and dummy variables, I drop the original columns from the data. I also drop highly collinear columns (NumberOfTime30-59DaysPastDueNotWorse and NumberOfTime60-89DaysPastDueNotWorse)¶

In [18]:
selected_features = [col for col in df if col.startswith('zipcode') or col not in ['PersonID', 'SeriousDlqin2yrs', 'zipcode', 'age', 'MonthlyIncome', 'NumberOfTime30-59DaysPastDueNotWorse', 'NumberOfTime60-89DaysPastDueNotWorse','NumberOfOpenCreditLinesAndLoans', 'NumberOfOpenCreditLinesAndLoans_discrete', 'MonthlyIncome_discrete', 'age_discrete']]
selected_y = 'SeriousDlqin2yrs'

Now I create train, test splits using these selected features¶

In [19]:
x_train, x_test, y_train, y_test = assignment2_functions.split_data(df, selected_features, selected_y, test_size=0.3)

Using these train/test splits, I build a decision tree classifier model. Without specifying a max_depth, the accuracy of the model is 80.7%. This means that the model predicts the y-variable serious delinquency in 2 years correctly 80.7% of the time.¶
